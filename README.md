# Portfolio Website Frontend

A modern, responsive, and interactive frontend for a personal portfolio website, built with cutting-edge web technologies. This project serves as a foundational template, showcasing a clean architecture, dynamic theming, and smooth animations.

## ‚ú® Features

*   **Dynamic Theme Toggling:** Seamlessly switch between light and dark modes with a visually engaging curtain animation.
*   **Modern Stack:** Built with React, TypeScript, and Vite for a fast and efficient development experience.
*   **Responsive Design:** Styled with Tailwind CSS for a utility-first approach, ensuring adaptability across various screen sizes.
*   **Smooth Animations:** Leverages GSAP (GreenSock Animation Platform) for high-performance, fluid UI transitions.
*   **Robust Testing:** Includes a comprehensive unit testing setup with Vitest and React Testing Library to ensure reliability and maintainability.
*   **Clean Architecture:** Organized with clear separation of concerns using custom hooks and a component-based structure.

## üì¶ Getting Started

Follow these instructions to get a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/en/) (which includes npm) installed on your system.

### Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/lokiwilded/React-Typescript-dark-mode-toggle-with-GSAP-animations-.git
    cd front
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

### Running the Project

*   **Development Mode:**
    Starts the development server with Hot Module Replacement (HMR).
    ```bash
    npm run dev
    ```
    Open your browser to `http://localhost:5173` (or the port indicated in your terminal).

*   **Build for Production:**
    Compiles the TypeScript code and bundles the application for production. The output will be in the `dist/` directory.
    ```bash
    npm run build
    ```

*   **Preview Production Build:**
    Serves the production build locally for testing.
    ```bash
    npm run preview
    ```

## ‚úÖ Running Tests

To execute the unit tests for the project:

```bash
npm run test
```

This will run all tests configured with Vitest and report the results in your terminal.
